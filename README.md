#  csgo查询工具

在
https://github.com/Cra2yQi/steam-check-recently-dropped
的基础上进行了修改，增加了检查个人信息、VAC状态的功能。

## python版本

```bash
    python 3.10.*
```

## install

```bash
    pip install -r requirements.txt
```

## usage

```bash
    python win_main.py
```

## feature

| 功能 | 说明 |
| :---: | :---: |
| 检查掉落 | 检查最近掉落的物品、掉落日期 |
| 检查个人信息 | 检查个人信息（等级、升级还需经验、本周挂箱状态）、VAC状态 |
